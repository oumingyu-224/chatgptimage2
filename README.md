# ChatGPT Image 2

ChatGPT Image 2 is an AI image generation website for creating, editing, downloading, and managing AI-generated images.

Website: [ChatGPT Image 2](https://www.chatgptimage2.app)

## Site Structure

### Public Pages

- `/` - Landing page and image generation entry
- `/create` - AI image creation page
- `/showcases` - Public showcase gallery
- `/pricing` - Pricing and credits page
- `/blog` - Blog index
- `/posts/[slug]` - Blog/article detail pages
- `/privacy-policy` - Privacy policy
- `/terms-of-service` - Terms of service

### User Area

- `/settings/profile` - User profile
- `/settings/credits` - Credit balance and credit records
- `/settings/billing` - Subscription and billing
- `/settings/payments` - Payment records
- `/settings/my-works` - User-generated image works
- `/settings/security` - Account security

### AI Activity

- `/activity/ai-tasks` - User AI generation task history
- `/activity/chats` - User chat history

### Admin Area

- `/admin` - Admin dashboard
- `/admin/users` - User management
- `/admin/roles` - Role management
- `/admin/permissions` - Permission management
- `/admin/categories` - Category management
- `/admin/posts` - Post management
- `/admin/prompts` - Prompt management
- `/admin/showcases` - Showcase management
- `/admin/payments` - Payment management
- `/admin/subscriptions` - Subscription management
- `/admin/credits` - Credit management
- `/admin/ai-tasks` - AI task management
- `/admin/chats` - Chat management
- `/admin/settings/general` - General settings
- `/admin/settings/auth` - Authentication settings
- `/admin/settings/payment` - Payment settings
- `/admin/settings/email` - Email settings
- `/admin/settings/storage` - Storage settings
- `/admin/settings/ai` - AI provider settings
- `/admin/settings/safety` - Safety and prompt moderation settings
- `/admin/settings/analytics` - Analytics settings
- `/admin/settings/ads` - Ads settings
- `/admin/settings/affiliate` - Affiliate settings
- `/admin/settings/customer_service` - Customer service settings

## Core Features

- Text-to-image generation
- Image-to-image generation
- Prompt moderation with DeepSeek
- User work history and image download
- Credit-based generation system
- Subscription and payment support
- Admin-managed AI provider configuration
- Admin-managed safety settings
- Public showcase gallery
- Blog and content management
- Role-based admin permissions

## Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- Drizzle ORM
- Better Auth
- PostgreSQL / compatible database
- Multi-provider AI generation integration
